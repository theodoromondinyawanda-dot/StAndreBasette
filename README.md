St. André Bessette Parish Website - Dandora
This is the official digital platform for St. André Bessette Parish in Dandora. The project provides a responsive client-side interface for parishioners and a robust administrative dashboard for parish management.
## Project Overview
The goal of this website is to bridge the gap between the parish and the community by providing real-time access to liturgical schedules, announcements, and sacramental information.
### Key Features
Client Side
Real-time Liturgical Calendar: Automated display of Mass times and special feast days.
Parish Groups (Jumuiyas): Dedicated sections for Small Christian Communities.
Responsive Design: Optimized for mobile users in the Dandora community.
Custom Notifications: Users receive alerts for urgent parish updates.
Admin Dashboard
Content Management: Easy-to-use interface for updating announcements without touching code.
Firebase Integration: Secure data handling using Firebase Realtime Database.
Loading States: Optimized UI for slow network conditions to ensure administrative efficiency.
### Technology Stack
Frontend: HTML5, CSS3 (Tailwind CSS), JavaScript.
Backend/Database: Firebase (Hosting & Realtime Database).
Deployment: Automated via GitHub Actions.
Development Environment: Developed entirely on Android via Termux.
